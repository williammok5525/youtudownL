# youtudownL
video download exe
由python转化，若有ffmpeg最好，没有ffmpeg也是ok的。
没有ffmpeg文件会保存成.webm格式
这种格式仍然可以打开并且观看
It is converted from python. It is best if ffmpeg is available, but it is also okay if ffmpeg is not available.
If no ffmpeg file is found, it will be saved as a .webm file.
This format can still be opened and viewed.
